# 💪 Fitness Tracker App (Stacked + Hive)

A modern Flutter-based workout and fitness tracking application built using **Stacked architecture** and **Hive database** for offline storage.

This app helps users track workouts, manage exercise routines, and monitor fitness progress efficiently.

---

## 🚀 Features

- 📱 Clean & Responsive UI
- 🏋️ Workout & Exercise Management
- 💾 Offline Data Storage using Hive
- 🧩 Stacked (MVVM) Architecture
- ⚡ Fast Performance
- 🔒 Local Data Security
- 📊 Progress Tracking

---

## 🛠️ Technologies Used

- Flutter (Latest Stable)
- Dart
- Stacked Architecture (MVVM)
- Hive Database (Local Storage)
